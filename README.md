# league

Read a file with results, and generate a league table.

## Build

On Mac/unix like

```./gradlew build```

## Run

To run the 

```java -jar build/libs/league-1.0.0.jar results.txt```

if you are happy setting up an alias

```
alias results="java -jar build/libs/league-1.0.0.jar"
results results.txt 
```
